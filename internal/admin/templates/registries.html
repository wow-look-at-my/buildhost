Registry Endpoints

Direct Downloads

Download artifacts directly by platform. Supports version pinning, latest, and branch-based resolution.

Latest{{.BaseURL}}/dl/{project}/latest/{os}/{arch}
Version{{.BaseURL}}/dl/{project}/{version}/{os}/{arch}
Branch{{.BaseURL}}/dl/{project}/branch/{branch}/{os}/{arch}
curl
curl -fsSL -H "Authorization: Bearer $TOKEN" \
  {{.BaseURL}}/dl/{project}/latest/linux/amd64 -o {project}
Query parameters
?format=raw       # Default binary (stripped if available)
?format=deb       # Debian package
?format=brew      # Homebrew bottle
?format=npm       # npm tarball
?debug=1          # Debug symbols

APT Repository

Debian/Ubuntu package repository. Packages are generated at publish time.

Release{{.BaseURL}}/apt/{project}/dists/stable/Release
InRelease{{.BaseURL}}/apt/{project}/dists/stable/InRelease
Packages{{.BaseURL}}/apt/{project}/dists/stable/main/binary-{arch}/Packages
Pool{{.BaseURL}}/apt/{project}/pool/{filename}
Setup (public project)
echo "deb [trusted=yes] {{.BaseURL}}/apt/{project} stable main" \
  | sudo tee /etc/apt/sources.list.d/{project}.list
sudo apt update && sudo apt install {project}
Setup (private project)
echo "deb [trusted=yes] {{.BaseURL}}/apt/{project} stable main" \
  | sudo tee /etc/apt/sources.list.d/{project}.list
cat <<EOF | sudo tee /etc/apt/auth.conf.d/{project}.conf
machine {{.BaseURL}}/apt/{project}
  login token
  password $TOKEN
EOF
sudo apt update && sudo apt install {project}

Homebrew Tap

Homebrew formula served as a single .rb file. Auto-detects macOS and Linux bottles.

Formula{{.BaseURL}}/brew/{project}.rb
Install (public project)
brew install {{.BaseURL}}/brew/{project}.rb
Install (private project)
HOMEBREW_BUILDHOST_TOKEN=$TOKEN brew install {{.BaseURL}}/brew/{project}.rb

npm Registry

npm-compatible registry. Packages are scoped under @buildhost.

Package metadata{{.BaseURL}}/npm/@buildhost/{project}
Tarball{{.BaseURL}}/npm/@buildhost/{project}/-/{project}-{version}.tgz
Setup
npm config set @buildhost:registry {{.BaseURL}}/npm/
npm config set //{{.BaseURL}}/npm/:_authToken $TOKEN   # if private
npm install @buildhost/{project}

OCI Distribution (Docker)

OCI-compatible registry for pulling artifacts as container images.

API check{{.BaseURL}}/v2/
Manifest{{.BaseURL}}/v2/{project}/manifests/{reference}
Blob{{.BaseURL}}/v2/{project}/blobs/{digest}
Docker pull
docker pull {{.BaseURL}}/{project}:{version}
Private project
echo $TOKEN | docker login {{.BaseURL}} -u token --password-stdin
docker pull {{.BaseURL}}/{project}:{version}

REST API

JSON API for managing projects, releases, and artifacts programmatically.

List projectsGET {{.BaseURL}}/api/v1/projects
Get projectGET {{.BaseURL}}/api/v1/projects/{project}
List releasesGET {{.BaseURL}}/api/v1/projects/{project}/releases
PublishPOST {{.BaseURL}}/api/v1/projects/{project}/publish
Authentication
# Bearer token
curl -H "Authorization: Bearer $TOKEN" {{.BaseURL}}/api/v1/projects

# Basic auth
curl -u "token:$TOKEN" {{.BaseURL}}/api/v1/projects

# Query parameter (for clients that can't set headers)
curl "{{.BaseURL}}/api/v1/projects?token=$TOKEN"
